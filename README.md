# Sign Language Detection

A machine learning project that detects sign language gestures from video input and converts them into text, helping deaf people communicate effectively.

## Features
- Real-time video input
- Recognizes multiple sign language gestures
- Displays corresponding word/text

## Tech Stack
- Python
- TensorFlow, Keras
- OpenCV
- NumPy, Pandas
- Matplotlib / Seaborn
